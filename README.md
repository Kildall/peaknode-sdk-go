# Peaknode Go SDK

Go client library for the Peaknode Proof of Reserves API.

## Installation

```bash
go get github.com/kildall/peaknode-sdk-go
```

## Usage

```go
import peaknode "github.com/kildall/peaknode-sdk-go"

client := peaknode.NewClient("https://api.peaknode.ar")
```
